২. আপনার টার্মিনাল/কমান্ড প্রম্পট (cmd) চালু করুন এবং এই ডিরেক্টরিটিতে যান:

## 📝 TypeScript-এ লুপ নিয়ে কিছু গুরুত্বপূর্ণ তথ্য

জাভাস্ক্রিপ্ট এবং টাইপস্ক্রিপ্ট-এ লুপের বেসিক লজিক বা সিনট্যাক্স একই। মূল পার্থক্য হলো **Type Safety (টাইপ সেফটি)**।

### ১. সাধারণ `for` লুপে টাইপ ডিক্লেয়ারেশন:
জাভাস্ক্রিপ্ট-এ আমরা লিখি:
```javascript
for (let i = 0; i < 5; i++) { ... }
```
TypeScript-এ আমরা লুপ কাউন্টার `i` এর টাইপ সরাসরি বলে দিতে পারি:
```typescript
for (let i: number = 0; i < 5; i++) { ... }
```

### ২. `for...of` লুপে অটোমেটিক টাইপ ডিডাকশন:
যদি আপনি একটি `string` অ্যারের ওপর লুপ চালান, TypeScript ওটি নিজে থেকেই সনাক্ত করে এবং প্রতিটি আইটেমকে `string` হিসেবে ট্রিট করে।
```typescript
const fruits: string[] = ["Apple", "Banana"];
for (const fruit of fruits) {
    // এখানে 'fruit' অটোমেটিকালি 'string' টাইপ পাবে।
    console.log(fruit.toUpperCase()); // সেফলি মেথড ব্যবহার করা যাবে
}
```

### ৩. `for...in` লুপে অবজেক্টের কী (Keys) অ্যাক্সেস করা:
টাইপস্ক্রিপ্ট-এ অবজেক্টের কী অ্যাক্সেস করতে হলে `keyof` কি-ওয়ার্ড ব্যবহার করতে হয়, যাতে অস্তিত্বহীন কোনো প্রোপার্টি রিড না হয়।
```typescript
interface User {
    name: string;
    age: number;
}
const user: User = { name: "Nabail", age: 21 };

for (const key in user) {
    // developer[key as keyof User] এভাবে অ্যাক্সেস করতে হয়
    console.log(`${key}: ${user[key as keyof User]}`);
}
```

## 🎯 আপনার জন্য কিছু প্র্যাকটিস চ্যালেঞ্জ (Practice Exercises)

আপনার প্র্যাকটিস করার জন্য `index.ts` ফাইলে নিচের প্রবলেমগুলো নিজে টাইপস্ক্রিপ্ট কোড লিখে সমাধান করার চেষ্টা করুন:

1. **জোড় সংখ্যা বের করা (Even Numbers):** ১ থেকে ২০ পর্যন্ত একটি লুপ চালান এবং শুধুমাত্র জোড় (even) সংখ্যাগুলো প্রিন্ট করুন। (লুপ কাউন্টারের টাইপ `number` ঘোষণা করুন)।
2. **অ্যারের যোগফল (Sum of Array):** একটি `number[]` অ্যারে তৈরি করুন `[5, 10, 15, 20]` এবং একটি `for...of` লুপের সাহায্যে সেগুলোর মোট যোগফল বের করুন।
3. **বিপরীত লুপ (Reverse Loop):** ১০ থেকে ১ পর্যন্ত উল্টো লুপ চালনা করে প্রিন্ট করুন (১০, ৯, ৮ ... ১)।
4. **নামতার লুপ (Multiplication Table):** যেকোনো একটি সংখ্যার নামতা (৫ এর ঘরের নামতা) `for` লুপ দিয়ে প্রিন্ট করুন।
